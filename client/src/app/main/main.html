Introdução

Tecnologias utilizadas

Back end


Java 8 - Linguagem

Maven - Gerênciador de dependências

Spring Boot - Scaffolding para back end, acelera o processo de configuração

Spring Test - Módulo de testes do Spring

Spring Web - Módulo com as dependências web para a aplição

Tomcat - Servidor de aplicação, neste caso incorporado pelo Spring Boot

Lombok - Biblioteca para geração de métodos getters e setters com a anotação @Data

Orika - Biblioteca para geração de métodos construtores

Spring Security - Módulo de segurança para aplicações

Spring Data/JPA - Módulo de mapeamento ao modelo relacional(ORM)

MySQL - Banco de dados

HSQLDB - Banco de dados em memória para testes


Front end


AngularJS - Framework para auxiliar na execução de single page applications

Yeoman -Scaffolding para front end

NPM (Node Package Manager) - Gerênciador de pacotes node

Bower - gerênciador de dependências client side

Gulp - Task runner

Coffeescript - linguagem que compila para javascript

Bootstrap - Framework para desenho de telas HTML, CSS

SASS - Linguagem que compila para CSS
